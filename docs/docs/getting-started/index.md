# Getting Started

Welcome to Muwanx! This guide will help you install and start using Muwanx to create interactive MuJoCo simulations in the browser.

## What is Muwanx?

Muwanx is a tool for creating interactive, browser-based MuJoCo simulations. It allows you to:

- Build and visualize MuJoCo physics simulations
- Create interactive web applications from Python or JavaScript
- Add AI policies to control robots
- Share your simulations easily through the web

## Quick Navigation

- **[Installation](installation.md)** - Install Muwanx using pip or npm

## Requirements

- **Python**: Version 3.10 or higher
- **Node.js**: Version 20 or higher (for npm installation)
- **Browser**: Modern browser with WebAssembly and WebGL support
